# How to learn golang

学习一门新语言，一般都是遵循先从基础语法开始学起，在尝试做练习题，最后进入实战实操一个上手项目做一个练手外加熟悉go语言的接口调用

1. 一个星期学习基础语法
2. 一两天做相关练习题
3. 一个星期实战实操一个简单的项目，最好这个项目能面面俱到

## go练习题

[题库点击这里](https://goquiz.github.io/)



## go的实战实操

实战项目：多人聊天室

考察点：

* 基本语法

* map、slice、struct、channel的使用

* goroutine协程的使用

* 网络通信的使用

* 心跳包的实现

* 文件/数据库数据储存

基本涵盖了go的所有重要的知识点，所以这是个很完美的实战项目。限期一个星期内完成！
